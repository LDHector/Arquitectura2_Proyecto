2.1 Sistemas de Numeración

+
Decimal (Base 10)

Dígitos: 0-9

12310 = 1×10² + 2×10¹ + 3×10⁰
Binario (Base 2)

Dígitos: 0,1

10102 = 1×2³ + 0×2² + 1×2¹ + 0×2⁰
Octal (Base 8)

Dígitos: 0-7

128 = 1×8¹ + 2×8⁰
Hexadecimal (Base 16)

Dígitos: 0-9, A-F

A316 = 10×16¹ + 3×16⁰

2.2 Conversión entre Sistemas

+
Decimal a Binario

División sucesiva entre 2

1310 = 11012
Binario a Decimal

Suma de potencias de 2

11012 = 8+4+0+1 = 1310
Binario a Hexadecimal

Agrupar en nibbles (4 bits)

1101 10102 = DA16

2.3 Representación de Enteros con Signo

+

Bit más significativo = signo (0=+, 1=-)

Ventajas: Fácil de entender

Desventajas: Dos ceros, operaciones complejas

+5 = 00000101, -5 = 10000101

Inversión bit a bit para negativos

Ventajas: Más simple que signo-magnitud

Desventajas: Sigue teniendo dos ceros

+5 = 00000101, -5 = 11111010

Complemento a 1 + 1, estándar actual

Ventajas: Un solo cero, operaciones simples

Desventajas: Rango asimétrico

+5 = 00000101, -5 = 11111011
