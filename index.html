Unidad I: Introducción a la Arquitectura y Organización

Fundamentos

1.1 Arquitectura y Organización de Computadoras

+

Arquitectura: atributos visibles al programador, que describen cómo se planifica y diseña el sistema.

Organización: trata sobre la implementación física, los componentes y su interconexión.

Perspectiva Funcional
  • Analiza la función de cada componente
  • Qué hace cada parte del sistema
  • Enfoque de comportamiento
Perspectiva Estructural
  • Estudia posición e interconexión
  • Cómo está construido el sistema
  • Enfoque de implementación

1.3 Máquina Secuencial de Programa Almacenado

+

Este modelo propone que tanto los datos como las instrucciones de un programa se almacenen en la misma memoria. La máquina ejecuta las instrucciones una por una mediante un dispositivo de control.

Requiere cinco unidades funcionales:

  • Órgano de cálculo (ALU o CPU)
  • Memoria
  • Unidad de control
  • Unidad de entrada
  • Unidad de salida

1.4 Organización de la Computadora

+

Se puede estudiar desde dos enfoques complementarios:

  • Funcional: describe las tareas que realiza la computadora: procesamiento, almacenamiento, traslado y control de datos.
  • Estructural: define los componentes físicos: CPU, memoria, y unidades de entrada/salida.

Ambos enfoques son esenciales para comprender cómo funciona una computadora moderna.

Unidad II: Representación de Datos y Aritmética Computacional

Fundamentos

2.1 Sistemas Numéricos

+

Las computadoras utilizan diferentes sistemas numéricos para representar datos:

Binario (Base 2)

Usa solo 0 y 1

10102 = 1010
Octal (Base 8)

Usa dígitos 0-7

128 = 1010
Hexadecimal (Base 16)

Usa 0-9 y A-F

A16 = 1010

2.2 Representación de Enteros

+

Bit más significativo representa el signo (0=+, 1=-)

+5 = 00000101, -5 = 10000101

Se invierten todos los bits para números negativos

+5 = 00000101, -5 = 11111010

Complemento a 1 + 1, más usado en computadoras

+5 = 00000101, -5 = 11111011

2.3 Aritmética de Punto Flotante

+

Representación de números reales usando notación científica:

Formato: Número = mantisa × BASEexponente

Estándar IEEE 754:

  • Simple precisión: 32 bits
  • Doble precisión: 64 bits
